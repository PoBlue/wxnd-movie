# 看看侃侃电影 🎥

微信学位最后一个项目

## 如何安装使用

1. 安装依赖
```bash
cd server && npm install
```

2. 打开 `微信开发者工具` -> 点击 `腾讯云` -> `上传测试代码`

## 应用介绍

- 热门电影列表
- 影评列表
- 语音 / 文字评论
- 收藏影评
